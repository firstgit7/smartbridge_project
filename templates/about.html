About the Project

The AI Powered Credit Card Approval Prediction System is an end-to-end machine learning application designed to automate and improve the accuracy of credit decisions.

How It Works

Data Processing: Financial history and demographics are cleaned, encoded, and scaled dynamically.

Machine Learning: We use an ensemble of models (like Random Forest) trained with SMOTE balancing to rigorously evaluate credit risk.

Real-time API: The Flask backend provides instant prediction results via secure REST APIs.
